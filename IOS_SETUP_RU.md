# Установка на личный iPhone

1. Откройте `ios/ObjectLens/ObjectLens.xcodeproj` в Xcode.
2. В Xcode войдите в Apple ID: Settings → Accounts.
3. В Signing & Capabilities выберите Personal Team.
4. Измените Bundle Identifier на уникальный, например `com.yourname.ObjectLens`.
5. Подключите iPhone кабелем, подтвердите доверие компьютеру.
6. На iPhone включите Developer Mode, если iOS попросит: Settings → Privacy & Security → Developer Mode.
7. Выберите физический iPhone в списке устройств Xcode.
8. Нажмите Run. При первом запуске выдайте доступ к камере.
9. Бесплатная подпись Apple ID ограничена сроком действия профиля и количеством приложений; при истечении срока повторите установку из Xcode.
